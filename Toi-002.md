## 問2
1から1000までの数字の内、7の倍数を出力するには以下のプログラムで満たす事ができる。
``` Java
    List<Integer> list = new ArrayList<Integer>();

    for(int j=1;j<=1000;j++) {
        if (j % 7 == 0) {
            list.add(j);
        }
    }

    System.out.println(list);
```

ただし、余りを出力する「％」は使用不可とする。

``` Java
            list.add(j);
        }
    }

```
